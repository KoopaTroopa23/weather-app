Weather App

A simple weather application built with **HTML, CSS, and JavaScript**.  
This project demonstrates DOM manipulation, event handling, and API integration.

---

Features

- Search weather by city name  
- Displays temperature and weather conditions  
- Uses real-time data from a weather API   when you enter your API into script.jss
// Your REAL API key from OpenWeatherMap
const apiKey = "Your api key here";
- Simple and responsive UI  
Change const url as well
const url = `web address here ${city}&appid=${apiKey}`;

---

# Tech Used 

- HTML  
- CSS  
- JavaScript  
- OpenWeatherMap API

How to use the ProjectL:

1. Clone or download this repository  
2. Open the project folder in your code editor  
3. Open `index.html` in your browser. Right click on index.html and click on "Open with Live Server" .  Note:  You must have VSCode extentsion.
